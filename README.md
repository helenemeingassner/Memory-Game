# Memory-Game
This program implements a simple memory game using an Arduino Uno. The system generates a sequence of colored LEDs, which the player must repeat by pressing the corresponding buttons. If the input is correct, the sequence gets longer; if not, the game resets. A 16×2 LCD (I²C) displays game status and feedback.
